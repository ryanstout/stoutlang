# StoutLang

## Learn

New to StoutLang? [Start Here](https://github.com/ryanstout/stoutlang/blob/master/docs/overview.md)

## Install

```shell
make setup
```

## Interpreter


```shell
./bin/run examples/hello_world.sl
```

## AOT compile

```shell
./bin/build {.sl file path} {output file path}

./bin/build examples/test2.sl test
```
